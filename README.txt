
ИНСТРУКЦИЯ

1. Telegram-бот
- Создай бота через @BotFather
- Получи TOKEN
- Вставь TOKEN в bot.py

2. Stripe
- Зарегистрируйся на stripe.com
- Получи SECRET_KEY
- Вставь ключ в server.py

3. Запуск
pip install flask python-telegram-bot stripe
python server.py
python bot.py

Заявки сохраняются в leads.csv
